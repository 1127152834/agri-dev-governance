# member-init 实施计划

## 1. 目标

为治理仓库补齐 `member-init` Skill 及其成员模板，使团队成员能够按统一结构完成个人信息、状态面板、任务容器和日报目录初始化。

## 2. 任务拆解

### 任务一：补齐设计文档

- 说明 `member-init` 的职责边界、输入输出和初始化目录结构。

### 任务二：编写成员初始化规范文档

- 新建 `docs/member-init.md`
- 定义初始化范围、目录结构、文件职责和使用方式。

### 任务三：创建模板

- 创建 `templates/member/` 下的成员模板文件。

### 任务四：编写 Skill

- 完成 `skills/member-init/SKILL.md`
- 引用模板目录和初始化结果。

### 任务五：验证与提交

- 运行 skill 校验脚本
- 检查模板与 Skill 是否一致
- 提交当前改动
