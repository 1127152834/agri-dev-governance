# 治理体系培训 Deck 实施计划

## 1. 目标

交付一份零依赖、单文件 HTML 培训演示稿，用于 40 分钟左右的治理体系培训。

## 2. 任务拆解

### 任务一：搭建单文件演示结构

- 创建 `docs/training/governance-training.html`
- 内联 CSS 和 JS
- 提供稳定的翻页、导航点、键盘控制

### 任务二：填充 14 页内容

- 按设计文档中的页纲填充每页内容
- 控制每页信息密度，避免超出视口

### 任务三：实现视觉与动画

- 深色专业主题
- 前半段建立治理权威感
- 中段步骤页重做为“战术卡”
- 轻量 reveal 动画
- 适配桌面演示和浏览器全屏

### 任务四：本地验证

- 浏览器打开演示稿
- 验证翻页、导航、布局和移动端基础适配

### 任务五：提交

- 提交 Deck HTML 与实施计划

## 3. 当前重构原则

- 不在旧版基础上做补丁式小修
- 重构为更强的培训叙事结构
- 优先保证可讲、可翻页、可全屏演示
