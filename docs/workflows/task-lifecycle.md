# 任务生命周期

## 主状态流

```text
草稿 -> 已发布 -> 已指派/已认领 -> 开发中 -> 待测试 -> 待评审 -> 已完成 -> 已归档
```

## 异常状态

- 已阻塞
- 已取消

## 状态说明

### 00-draft

项目经理仍在拆分或整理任务，尚未进入可认领状态。

### 01-ready

任务已发布，可被认领或指派。

### 02-assigned

任务已明确责任人，但尚未正式进入开发。

### 03-in-progress

任务已进入开发执行阶段。

### 04-waiting-test

开发已完成，等待验证。

### 05-waiting-review

测试通过，等待评审。

### 06-done

任务已完成，可进入归档准备。

### 07-blocked

任务因依赖、环境或需求不清等原因阻塞。

### 08-cancelled

任务被取消，但必须保留完整记录。

### 99-archived

历史任务归档目录。
